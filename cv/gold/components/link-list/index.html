The .gov.au means it’s official

Australian Government websites always use a .gov.au domain. Before sharing sensitive information online, make sure you’re on a .gov.au site by inspecting your browser’s address (or 'location') bar.

This site is secure.

The https:// ensures that you are connecting to the official website and that any information you provide is encrypted and transmitted securely.

Link listv1.0.0

The link list style is a simple list of vertical or horizontal links used for site navigation. It is used to order information for users.

#

Default

The default link list component removes the normal bullets and spacing associated with a list.

  • HTML
  • React
<ul class="au-link-list">
  <li><a href="#">Options</a></li>
  <li><a href="#">Help</a></li>
  <li><a href="#">Sign Out</a></li>
</ul>

import AUlinkList from '@gov.au/link-list';

<AUlinkList items={[
  {
    link: '#',
    text: 'Options',
  },
  {
    link: '#',
    text: 'Help',
  },
  {
    link: '#',
    text: 'Sign Out',
  },
]} />
#

Inline

The Inline link list is simply a modifier to the default link list which makes the links run inline/horizontally.

  • HTML
  • React
<ul class="au-link-list au-link-list--inline">
  <li><a href="#">Options</a></li>
  <li><a href="#">Help</a></li>
  <li><a href="#">Sign Out</a></li>
</ul>

import AUlinkList from '@gov.au/link-list';

<AUlinkList inline items={[
  {
    link: '#',
    text: 'Options',
  },
  {
    link: '#',
    text: 'Help',
  },
  {
    link: '#',
    text: 'Sign Out',
  },
]} />
