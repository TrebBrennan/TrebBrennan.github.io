Colour

Our design system uses colour in a range of meaningful and intentional way to create patterns and visual cues. We offer 5 colour palettes:

• main palette
• contrast main palette
• dark palette
• contrast dark palette
• notification palette

These 5 palettes work together to support a variety of uses cases and provide a consistent look for action or link colours and focus states regardless of the palette combination./

Our colour palettes have also been designed to be implemented in a themable manner. The universal colour variables are determined by common roles and usage; they are not based singularly on a colour value (ie unique hex code). The same colour value may be assigned to multiple variables in a theme's palette when the values have distinctly different roles.

Soft Buttons

Soft buttons place less visual emphasis on the button. Soft buttons are useful when grouping buttons with differing importance.
